Access Teaching Materials

  • Article
  • 07/01/2023

In this article

As part of Microsoft Learn for Educators (MSLE) program, you have access to the Microsoft Official Curriculum. Review materials and prepare for your course with the official Microsoft learning products available in the Learning Download Center, via MSLE portal.

In the Learning Download Center, you’ll find:  

  • Microsoft Official Curriculum (MOC): course power point and trainer prep guide 
  • Educator teaching guide: educator specific course information to prepare for delivery
  • Assessment guide: formative and summative assessments for students; hands-on activity

Steps to access the Teaching Materials

  1. Go to the MSLE Portal, and sign in with your Microsoft Account.
  2. On the left-hand navigation panel, select Learning Download Center.
  3. If not already, sign in using your personal email address.
  4. If already signed-in, accept the terms & conditions.
  5. Access the educator resources via the Learn for Educators program.
  6. Expand the Learn for Educators program and select files/folders to be added to Downloads Queue.
  7. Select the Downloads Queue and click Download - if prompted install the Download Manager to download files.

A tutorial on how to access the materials can be viewed in the, Introduce the Learning Download Center video.
