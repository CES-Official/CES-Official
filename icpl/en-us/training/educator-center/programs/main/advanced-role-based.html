Microsoft Advanced Role-Based

  • Article
  • 04/05/2024

In this article

Microsoft Advanced Role-Based curriculum provides associate level knowledge of Microsoft cloud and business application services. They're ideal for students looking to begin learning valuable job role skills.

Microsoft Azure Administrator

This course covers topics like managing Azure identities and governance, implementing and managing storage, deploying and managing Azure compute resources, configuring and managing virtual networking, and monitoring and backing up Azure resources. Completion of the course prepares students for the AZ-104: Microsoft Azure Administrator certification exam. Candidates for this exam should have a strong understanding of core Azure services, Azure workloads, security, and governance.

Students may be eligible for ACE college credit if they pass this certification exam. See ACE college credit for certification exams for details.

Designing and Implementing a Microsoft Azure AI Solution

This course covers topics like designing and implementing a Microsoft Azure AI solution; building, managing, and deploying AI solutions that leverage Azure Cognitive Services; Azure Cognitive Search; and Microsoft Bot Framework. Completion of the course prepares students for the AI-102: Designing and Implementing a Microsoft Azure AI Solution certification exam. Candidates for this exam should have a strong understanding of the components that make up the Azure AI portfolio, the available data storage options, and be able to apply responsible AI principles.

Students may be eligible for ACE college credit if they pass this certification exam. See ACE college credit for certification exams for details.

Designing and Implementing Microsoft DevOps Solutions

This course provides the knowledge and skills to design and implement DevOps processes and practices. Students will learn how to plan for DevOps, use source control, scale Git for an enterprise, consolidate artifacts, design a dependency management strategy, manage secrets, implement continuous integration, implement a container build strategy, design a release strategy, set up a release management workflow, implement a deployment pattern, and optimize feedback mechanisms. Completion of this course prepares students for the AZ-400: Designing and Implementing Microsoft DevOps Solutions certification exam.

Microsoft Azure Security Technologies

This course covers topics like implementing security controls, maintaining an organizationâ€™s security posture, and identifying and remediating security vulnerabilities. This course includes security for identity and access, platform protection, data and applications, and security operations. Completion of the course prepares students for the AZ-500: Microsoft Azure Security Technologies certification exam. Candidates for this exam should have subject matter expertise implementing Azure security controls that protect identity, access, data, applications, and networks in cloud and hybrid environments as part of an end-to-end infrastructure.

Students may be eligible for ACE college credit if they pass this certification exam. See ACE college credit for certification exams for details.

Microsoft Power BI Data Analyst

This course discusses the various methods and best practices that are in line with business and technical requirements for modeling, visualizing, and analyzing data with Power BI. The course will also show how to access and process data from a range of data sources including both relational and non-relational data. This course also explores how to implement proper security standards and policies across the Power BI spectrum including datasets and groups. The course will also discuss how to manage and deploy reports and dashboards for sharing and content distribution. Completion of the course prepares students for the PL-300: Microsoft Power BI Data Analyst certification exam.

Microsoft Power Platform Developer

In this course students expect to learn how to design, develop, test, secure, and troubleshoot Microsoft Power Platform solutions. Students will implement components of a solution that include application enhancements, custom user experiences, system integrations, data conversions, and custom process automation. Completion of this course prepares students for the PL-400: Microsoft Power Platform Developer certification exam.

Designing and Implementing a Data Science Solution on Azure

This course teaches you to leverage your existing knowledge of Python and machine learning to manage data ingestion and preparation, model training and deployment, and machine learning solution monitoring in Microsoft Azure. This course will also show you how to operate machine learning solutions at cloud scale using Azure Machine Learning. Completion of the course prepares students for the DP-100: Designing and Implementing a Data Science Solution on Azure certification exam.

Students may be eligible for ACE college credit if they pass this certification exam. See ACE college credit for certification exams for details.

Data Engineering on Microsoft Azure

In this course, students will learn about the data engineering patterns and practices as it pertains to working with batch and real-time analytical solutions using Azure data platform technologies. The students will learn how to interactively explore data stored in files in a data lake. Completion of the course prepares students for the DP-203: Data Engineering in Azure certification exam

Students may be eligible for ACE college credit if they pass this certification exam. See ACE college credit for certification exams for details.

Developing Solutions for Microsoft Azure

This course teaches developers how to create solutions that are hosted in, and utilize, Azure services. The course covers topics like creating Azure App Services, developing storage solutions, and implementing secure cloud solutions. Completion of the course prepares students for the AZ-204: Developing solutions for Microsoft Azure certification exam.

Students may be eligible for ACE college credit if they pass this certification exam. See ACE college credit for certification exams for details.

Microsoft Power Platform Functional Consultant

This course provides students with the skills and knowledge required to successfully build and manage solutions built using Microsoft Power Platform. The course includes full coverage of the core Power Platform features such as Power Apps, Power Automate, Power Virtual Agents, Power Pages and Microsoft Dataverse. Completion of the course prepares students for the PL-200: Microsoft Power Platform Functional Consultant certification exam.

Microsoft Identity and Access Administrator

This course provides knowledge and skills on identity and access management in the cloud using Microsoft Entra ID and its related components in Azure. Completion of the course prepares the student for the SC-300: Microsoft Identity and Access Administrator certification exam.

Administering Information Protection and Compliance in Microsoft 365

In this course students learn how to protect information in your Microsoft 365 deployment. This course focuses on data lifecycle management and information protection within an organization. The course covers implementation of data loss prevention policies, sensitive information types, sensitivity labels, data retention policies and Office 365 message encryption among other related topics. Completion of the course prepares the student for the SC-400: Microsoft Information Protection Administrator certification exam.

Microsoft Dynamics 365 Business Central Functional Consultant

This course teaches you how to implement and configure Microsoft Dynamics 365 Business Central including setting up core functionality and modules and migrating data to Business Central for small and medium businesses. Completion of the course prepares students for the MB-800: Microsoft Dynamics 365 Business Central Functional Consultant certification exam.

Microsoft Dynamics 365 Customer Service

This course provides students with the skills and knowledge required to successfully implement and manage the Dynamics 365 Customer Service application. The course includes full coverage of the core Customer Service features such as Cases, Knowledge Management, SLAs, Entitlements, Omnichannel, and more. In addition, the course includes sections on using IoT with Connected Customer Service, and scheduling service activities. Completion of the course prepares student for the MB-230: Microsoft Dynamics 365 Customer Service certification exam.

Microsoft Dynamics 365 Sales Functional Consultant

This course provides students with the skills and knowledge required to successfully take a role as a Dynamics 365 Sales Functional Consultant. The course includes coverage of configuring Dynamics 365 Sales, managing cores sales tables, and configuring additional tools and services. Completion of the course prepares students for the MB-210: Microsoft Dynamics 365 Sales Functional Consultant certification exam.

Students may be eligible for ACE college credit if they pass this certification exam. See ACE college credit for certification exams for details.

Microsoft Dynamics 365 Marketing

This course reviews the marketing application configuration needed to drive business growth. It dives into lead management, marketing forms and pages, segmentation, and email marketing messages. All these pieces are tied together through interactive customer journey design, including event and survey configuration. Completion of the course prepares students for the MB-220: Microsoft Dynamics 365 Marketing certification exam.

Students may be eligible for ACE college credit if they pass this certification exam. See ACE college credit for certification exams for details.

Designing Microsoft Azure Infrastructure Solutions

This course focuses on how to demonstrate subject matter expertise in designing cloud and hybrid solutions that run on Azure. Students will learn how to design identity, governance, and monitoring solutions; design data storage solutions; design business continuity solutions; and design infrastructure solutions. Completion of this course prepares students for the AZ-305: Designing Microsoft Azure Infrastructure Solutions exam.

Administering Windows Server Hybrid Core Infrastructure

This course focuses on how to demonstrate subject matter expertise in configuring and managing Windows Server on-premises, hybrid, and infrastructure as a service (IaaS) platform workloads. Completion of this course prepare students for the AZ-800: Administering Windows Server Hybrid Core Infrastructure exam.

Administering Microsoft Azure SQL Solutions

This course focuses on how to demonstrate subject matter expertise in building database solutions that are designed to support multiple workloads built with SQL server on-premises and Azure SQL services. Completion of this course prepares students for the DP-300: Administering Microsoft Azure SQL Solutions exam.
